# jQuery Smooth Scrolling
**Smooth scrolling to page anchors**

## Usage
```js
$('html').smoothScrolling()
```

### Options
```js
$('html').smoothScrolling({
  offset: 10, // Default - 0
  speed: 1500, // Default - 1000

  // Sticky header
  headerClass: 'header', // Default - null
  headerFixedClass: 'header_fixed' // Default - null
})
```
